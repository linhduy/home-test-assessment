# home_test
## Clone source &nbsp;
```
$ git clone https://github.com/linhduy/home-test-assessment.git
```

## Installation &nbsp;
```
$ sudo npm install sails -g
```
## Setup

go to source and type

```
$ sudo npm install
```

## Mongodb config &nbsp;
create db: home_test

create dbuser: admin/admin

## Run

```
$ sails lift
```


## Test

* Use postman to test
ex: http://localhost:1337/user/login
